My Repositories

Overview

Here is a list of some of the projects I've worked on along with a brief description:
1. ScatterGather System: A virtual device driver that was programmed in C. The running code
simulates an online storage service like Box or Google Drive.
2. Malloc Lab: Developed a dynamic storage allocator i.e. a custom version of the malloc function
3. Computer Vision CNN Project: Coded the forward propagation of a CNN for image recognition using
MATLAB. Also generated statistics about the accuracy of the CNN.
4. Computer Vision Perspective Projection Project: MATLAB project that implements forward and inverse
camera projection using 3D scene coordinates. Generates error statistics as well.
5. Computer Vision Video Change Detection Project: MATLAB project to compare 4 different video change
detection algorithms side by side in different scenes.
